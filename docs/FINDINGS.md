## 1. Substrate `𝓘` (sims 1-7)

### Verified axioms (`sim_association_scheme.py`)

15/15 acceptance checks pass:
- Symmetric Bose–Mesner basis `{A₀, ..., A₈}` with `A₀ = I` and `Σ A_i = J`

### P-polynomial structure (`sim_p_polynomial_structure.py`)

Each `A_i = p_i(A₁)` for an explicit polynomial of degree exactly i.
The 9-dim Bose–Mesner algebra is generated by `A₁` alone.

### Honest negative findings

- `sim_tridiagonal_L1.py` — V_600 is NOT a narrow distance-regular graph.
  The intersection array `{1, 3, 5, 2, 5, 3, 5, 12; 12, 5, 3, 5, 2, 5, 3, 1}` does not

  with valencies (1, 12, 32, 42, 32, 1), not 9. Also not strict-DRG:

## 2. Closure operator `𝒞 = C_φ` (sim 8)

### Definition

```
C_φ := L_V₆₀₀ + φ⁻²·I = (12 + φ⁻²)·I − A₁
```

### Spectrum (`sim_closure_sigma_action.py`)

C_φ commutes with `A₁` and shares eigenspaces. On each `E_j`, C_φ acts as
scalar `μ_j = (12 + φ⁻²) − λ_j`. All 9 measured scalars match prediction
to machine precision.

### Galois conjugate operator

`C_{1-φ} = L + φ²·I` is the σ-Galois conjugate of `C_φ`. Acts on `E_j` with
scalar `(12 + φ²) − λ_j`. Numerical match exact at each eigenspace.

(C_φ, C_{1-φ}) is a σ-pair of distinct operators on `ℂ[V_600]`.

## 3. σ-pairing structure (sim 8)

### σ-fixed eigenvalues (in ℤ ⊂ ℤ[φ])

`{12, 3, 0, −2, −3}`. The eigenspaces `E_0, E_3, E_4, E_5, E_7` are
σ-invariant as sets.

### σ-paired (Galois-conjugate) eigenvalues

- `σ(6φ) = 6−6φ`: pairs `E_1 ↔ E_8` (each dim 4)
- `σ(4φ) = 4−4φ`: pairs `E_2 ↔ E_6` (each dim 9)

### The 94 σ-fixed dimensions

```

           = 1 + 16 + 25 + 36 + 16
           = 94
```

Matches the existence-life-closure programme's τ_ico convention exactly.

### Antipodal pairing of (E_3, E_7)

The dim-16 eigenspaces `E_3` (λ=+3) and `E_7` (λ=−3) are NOT σ-paired (both
σ-fixed), but ARE antipodal-paired: the antipodal operator `A_8` acts as
`−I` on `E_3` and `+I` on `E_7`. This is a separate ℤ_2-grading.

### Embedding note

In the standard Wikipedia even-permutation embedding of V_600 in ℝ⁴, the
coordinate-wise Galois action does NOT preserve V_600 as a set of points
(only the 8 axis + 16 cube vertices are fixed; 96 snub vertices map to odd
permutations). σ-pairing is therefore a SPECTRAL phenomenon in this
embedding, not a vertex permutation. The icosian ring 𝓘 itself is closed
under σ.

## 4. Generation operator `𝒢 = N_H` (sims 9, 10)

### Definition

```
N_H : 𝓘 → ℤ[φ]
N_H(q) := q·q̄ = q₀² + q₁² + q₂² + q₃²
```

### Surjectivity onto totally positive ℤ[φ]-primes (`sim_generation_operator.py`)

Every totally positive ℤ[φ]-prime π with rational norm ≤ 50 has an icosian
witness q ∈ 𝓘 (Lipschitz lattice, coefficients |a|,|b| ≤ 4) such that
`N_H(q) = π`.

Verified for 20 primes:

- 1 ramified above 5: π = 2 + φ
- 5 split-pair families above 11, 19, 29, 31, 41 (each with π and σ(π))

### σ-equivariance (`sim_triad_isomorphism.py`)

```
N_H(σ̂(q)) = σ(N_H(q))   for all q ∈ 𝓘
```

Verified on all 20 witness quaternions. Consequences:
- Inert primes admit σ̂-fixed (rational-coordinate Lipschitz) witnesses.
  Verified 9/9.
- Split-prime σ-pairs (π, σ(π)) correspond to σ̂-conjugate icosian pairs.
  Verified 5/5.
- Ramified prime 2+φ corresponds to icosian witness with explicit σ-twisted
  structure.

## 5. Representation numbers `r(π)` (sims 11, 12, 13)

### Closed form (`sim_representation_numbers.py`)

For every odd totally positive ℤ[φ]-prime π:

```
r(π) := #{q ∈ 𝓘_max : N_H(q) = π} = 120 · (1 + N_Q(π))     [maximal order]
```

Verified for EVERY prime type via complete Fincke–Pohst enumeration in the
maximal icosian order (`sim_theta_maximal_order.py`): split (N=11,19,29),
inert 2 (r=600) and 3 (r=1200), ramified 5 (r=720) — all r/120 = 1+N.

**No anomaly at 2 (corrected, 2026-06-08).** The maximal order gives
`r(2) = 600 = 120·5` — the standard Eisenstein value. The earlier `r(2)=24`
came from the **Lipschitz suborder** `𝓘_lip = ℤ[φ]^4` (8 units), a
bounded-range enumeration proxy that is non-maximal at 2. That suborder value,
and the local-2 correction it induced, are withdrawn.

### Multiplicativity (`sim_multiplicativity.py`)

For coprime totally positive `α, β ∈ ℤ[φ]`, the normalized count
`r̃ = r/120` is multiplicative: `r̃(α·β) = r̃(α)·r̃(β)`. Verified for 6
coprime pairs (inert×inert, inert×ramified, inert×split, split×split-σ,
split×split distinct-rationals); composite norm up to 209.

### L-function identification (`sim_theta_maximal_order.py`)

```
L(Θ_𝓘, s) := Σ_{α≠0 mod units} (r(α)/120) · N_Q(α)^(-s)
           = ζ_K(s) · ζ_K(s-1)        EXACTLY  (C_2 = 1)
```

where K = ℚ(√5). The inert-2 powers give r((2)^k)/120 = σ_1((2)^k) =
1,5,21,85, whose generating function is the standard Eisenstein local factor
`1/[(1-4^{-s})(1-4^{1-s})]` — so there is no local-2 correction.

Via Dedekind factorisation:
```

```

so the icosian-derived L-function admits the **Euler bridge factorisation**:
```
L(Θ_𝓘, s) = ζ(s) · L(s, χ_5) · ζ(s-1) · L(s-1, χ_5)        (exact)
```
